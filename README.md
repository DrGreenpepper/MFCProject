# 给同学写的程序

已挂GPL,请遵守

## 环境与功能
`Visual Studio 2017，MFC基础类库`

实现会员基本信息的创建与更新.

实现历史操作的记录查询.

简单的增删改查,提供了浮点类型的资金和整形的次数作为主要的修改内容.

## 介绍
> 起因：XXX同学与同学合资搞了个桌游店，需要一个会员管理系统，所以深夜一点委托我帮他搞一个。于是花了三天写了一个。
> 一天过去了，XXX同学没有来揍我，说明程序应该运行良好。
## 简介 

>默认登录账号密码都为 admin,Realease文件夹中有相关测试用数据.

>使用的是动态链接库，所以可以下载Realease文件夹中查看最后交给他的版本。

`.idx` ：读取的目录

`.swp` ：写缓存文件的东西

`.opt` ：用来检测首次使用的

`data` ：数据文件。
## 总结
  写的很垃圾，线性数据结构，没用数据库，高耦合低聚合，考虑到使用范围只是本地，随便算了算复杂度，在10w的会员数下，响应时间应该不会超过0.1s，嗯，能用就行。
  
  想他们的会员数也不会超过10W吧，超过了就让他们自己买程序去 （ ^w^）
